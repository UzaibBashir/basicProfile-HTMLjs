# Portfolio Website

A personal portfolio site built with HTML, Tailwind CSS, and JavaScript.

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- JavaScript

## Run Locally

1. Clone or download this repository.
2. Open `index.html` in your browser.

## Project Structure

- `index.html` - Main website file
- `app.js` - JavaScript interactions
- `img/` - Images and assets

## Customize

- Update personal info, skills, and links in `index.html`.
- Replace images inside `img/`.
